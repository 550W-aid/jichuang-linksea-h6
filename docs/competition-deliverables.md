# 交付物清单

## 技术文档

按赛题要求准备 `Word + PDF` 两个版本，建议结构：

1. 项目简介
2. 设计方案详述
3. 项目创新点
4. 测试结果
5. 团队介绍和竞赛收获

建议在封面或目录后的前 `1-2` 页先放亮点预览：

- 国产 FPGA 实时视频链路
- OV5640 到 VGA 的完整路径
- 低照增强与巡检叠加效果
- 帧率、资源占用和实时调参能力

## PPT

结构与技术文档保持一致，但强调可视化：

- 一页展示系统框图
- 一页展示板卡和接线图
- 一页展示算法前后对比
- 一页展示指标和资源
- 一页展示创新场景

## 演示视频

- 格式：`MP4`
- 方向：优先横屏
- 时长：控制在 `5 分钟` 内
- 必含内容：系统介绍、功能演示、关键指标、效果对比

## 源码压缩包

建议按以下目录打包：

- `fpga/`
- `mcu/`
- `tools/`
- `docs/`

同时附带：

- 使用说明
- 版本说明
- 已知限制

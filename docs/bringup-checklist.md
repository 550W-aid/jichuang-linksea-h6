# Bring-Up Checklist

## 环境准备

- [ ] 运行 `python tools/check_startup.py`
- [ ] 安装 `eLinx Design Suite`
- [ ] 安装 USB-JTAG 驱动
- [ ] 安装 USB-UART 驱动
- [ ] 安装 `STM32CubeIDE` 或 `STM32CubeCLT`
- [ ] 安装 `Git`
- [ ] 安装 `Python 3`
- [ ] 安装 Python 包 `pyserial`
- [ ] 将板卡资料归档到 `reference/`
- [ ] 准备 VGA 显示器和线缆
- [ ] 准备摄像头模组和排线

## 板级最小验证

- [ ] 板卡上电正常，电源指示灯状态正确
- [ ] JTAG 可以识别 FPGA 设备
- [ ] 工程可成功综合、布局布线、下载
- [ ] LED 闪烁逻辑正常
- [ ] 按键输入可被 FPGA 读取
- [ ] UART 回环或状态打印正常
- [ ] VGA 彩条正常显示

## 摄像头链路验证

- [ ] `OV5640` SCCB 初始化成功
- [ ] 传感器输出分辨率固定为 `640x480`
- [ ] 采集到的图像色彩正确
- [ ] 无行错位、无明显雪花、无黑屏
- [ ] 断开重连摄像头后可恢复

## 视频缓存与显示验证

- [ ] 帧缓存写侧正常
- [ ] 帧缓存读侧正常
- [ ] 连续运行 30 分钟无花屏
- [ ] 模式切换无死机
- [ ] FPS 计数正常

## 算法联调

- [ ] 灰度模式可切换
- [ ] 低照增强参数可调
- [ ] 边缘叠加可切换
- [ ] 直方图均衡化通路接通
- [ ] 缩放模块通路接通
- [ ] 旋转模块通路接通

## MCU 联调

- [ ] STM32 能写 FPGA 控制寄存器
- [ ] STM32 能读 FPGA 状态寄存器
- [ ] 菜单或按键逻辑可切换模式
- [ ] 默认参数可一键下发

## 交付前确认

- [ ] 完成文档截图和效果图采集
- [ ] 将证据同步归档到 `evidence/`
- [ ] 记录资源利用率
- [ ] 记录关键模式帧率
- [ ] 录制演示视频
- [ ] 打包源码、文档、PPT

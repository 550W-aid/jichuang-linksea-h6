# 控制寄存器映射

当前寄存器采用 `8-bit 地址 + 16-bit 数据` 形式，既适合早期串口联调，也便于后续挂到 `FSMC` 或板上互联总线。

## 地址表

| 地址 | 名称 | 访问 | 说明 |
|---|---|---|---|
| `0x00` | `mode` | R/W | 工作模式，`0=直通/彩条`，`1=灰度`，`2=低照增强`，`3=巡检叠加` |
| `0x01` | `algo_enable` | R/W | 算法开关位图 |
| `0x02` | `brightness_gain` | R/W | 低照增强增益，`0x0010=1.0x` |
| `0x03` | `gamma_sel` | R/W | Gamma/LUT 档位，`0-3` |
| `0x04` | `scale_sel` | R/W | 缩放模式选择 |
| `0x05` | `rotate_sel` | R/W | 旋转模式选择 |
| `0x06` | `edge_sel` | R/W | 巡检边缘叠加配置 |
| `0x07` | `osd_sel` | R/W | OSD 叠加配置 |
| `0x08` | `status` | RO | 状态位，见下文 |
| `0x09` | `fps_counter` | RO | 最近 1 秒帧数统计 |
| `0x0A` | `heartbeat` | RO | 控制域心跳计数 |

## `algo_enable` 位定义

| 位 | 名称 | 说明 |
|---|---|---|
| `0` | `gray_en` | 灰度/颜色空间变换 |
| `1` | `hist_eq_en` | 直方图均衡化 |
| `2` | `scale_en` | 固定倍数缩放 |
| `3` | `rotate_en` | 固定角度旋转 |
| `4` | `lowlight_en` | 低照增强 |
| `5` | `edge_en` | 边缘叠加 |

## `status` 位定义

| 位 | 名称 | 说明 |
|---|---|---|
| `0` | `uart_alive` | UART 控制模块在线 |
| `1` | `cam_init_done` | 摄像头初始化完成 |
| `2` | `frame_locked` | 帧缓冲稳定锁定 |
| `3` | `video_active` | 视频流处于活动状态 |

## UART 调试协议

主机到 FPGA 的命令帧固定为 6 字节：

| 字节 | 含义 |
|---|---|
| `0` | 帧头 `0x55` |
| `1` | 命令：`0x01=写`，`0x02=读`，`0x03=ping` |
| `2` | 寄存器地址 |
| `3` | 数据高字节 |
| `4` | 数据低字节 |
| `5` | 校验，前 5 字节按 XOR 计算 |

FPGA 返回 6 字节响应帧：

| 字节 | 含义 |
|---|---|
| `0` | 帧头 `0xAA` |
| `1` | 状态：`0x00=成功`，`0xE0=校验错误`，`0xE1=命令错误`，`0xE2=地址错误` |
| `2` | 寄存器地址 |
| `3` | 数据高字节 |
| `4` | 数据低字节 |
| `5` | 校验，前 5 字节按 XOR 计算 |
